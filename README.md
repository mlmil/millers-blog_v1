## How to Use

1. Open `millers_blog.html` in any web browser
2. All images and styling are self-contained

## Publishing

To publish this to GitHub Pages:

1. Create a new repository on GitHub called `millers-blog`
2. Run these commands in this folder:
   ```
   git add .
   git commit -m "Initial blog commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/millers-blog.git
   git push -u origin main
   ```
3. Enable GitHub Pages in repository settings
